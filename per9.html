Rumus Transformasi

1. Dilatasi (Refleksi)

Dilatasi, sebagaimana dijelaskan dalam materi, merujuk pada transformasi refleksi atau pencerminan yang membuat citra cermin dari suatu objek.

a. Refleksi terhadap sumbu x (garis y=0):

Transformasi membuat nilai x sama tetapi membalikkan nilai y berlawanan dengan posisi koordinat. Matriks transformasinya adalah:

[ 1 0 ] [ 0 -1 ]

b. Refleksi terhadap sumbu y (garis x=0):

Refleksi terhadap sumbu y membalikkan koordinat dengan nilai y tetap. Matriks transformasinya adalah:

[ -1 0 ] [ 0 1 ]

c. Refleksi terhadap titik pusat (Origin):

Menurut materi, ini adalah "Dilatasi terhadap sumbu x dan y sekaligus" dan "sama dengan rotasi 180° pada bidang xy". Matriks transformasinya adalah:

[ -1 0 ] [ 0 -1 ]

2. Shear (Geseran)

Shear adalah transformasi yang menyebabkan distorsi pada bentuk suatu objek, seolah-olah terjadi pergeseran antar sisinya.

a. Shear Terhadap Sumbu X (X-Shear):

X-Shear menggeser titik-titik secara horizontal sebanding dengan jarak vertikalnya dari sumbu x. Koordinat y tetap, sedangkan koordinat x berubah. Transformasi ini dinyatakan dengan matriks:

[ 1 shx ] [ 0 1 ]

Dengan koordinat transformasi:

x' = x + shx * y

y' = y

Parameter shx adalah faktor shear pada arah x.

b. Shear Terhadap Sumbu Y (Y-Shear):

Y-Shear menggeser titik-titik secara vertikal sebanding dengan jarak horizontalnya dari sumbu y. Koordinat x tetap, sedangkan koordinat y berubah. Transformasi ini dinyatakan dengan matriks:

[ 1 0 ] [ shy 1 ]

Dengan koordinat transformasi:

x' = x

y' = y + shy * x

Parameter shy adalah faktor shear pada arah y.

Penjelasan Hasil Visual Interaktif

1. Memahami Dilatasi (Refleksi) melalui Visualisasi

Ketika Anda memilih opsi "Refleksi (Dilatasi)" pada menu "Jenis Transformasi", visualisasi akan menunjukkan bagaimana objek asli dicerminkan terhadap sumbu yang Anda pilih.

  • Refleksi terhadap Sumbu X: Jika Anda memilih "Sumbu X" sebagai sumbu refleksi, Anda akan melihat bahwa setiap titik pada objek asli (misalnya (x, y)) akan dipetakan ke titik baru (x, -y). Nilai koordinat x tetap, sementara nilai koordinat y menjadi negatifnya. Objek merah akan menjadi cerminan objek biru terhadap sumbu horizontal (sumbu X).
  • Refleksi terhadap Sumbu Y: Jika Anda memilih "Sumbu Y", setiap titik (x, y) akan dipetakan ke (-x, y). Nilai koordinat y tetap, sementara nilai koordinat x menjadi negatifnya. Objek merah akan menjadi cerminan objek biru terhadap sumbu vertikal (sumbu Y).
  • Refleksi terhadap Titik Pusat (Origin): Jika Anda memilih "Titik Pusat", setiap titik (x, y) akan dipetakan ke (-x, -y). Kedua koordinat, x dan y, akan menjadi negatifnya. Ini setara dengan rotasi 180 derajat mengelilingi titik pusat.

2. Memahami Shear (Geseran) melalui Visualisasi

Ketika Anda memilih opsi "Shear X" atau "Shear Y", visualisasi akan mendemonstrasikan bagaimana transformasi shear mendistorsi bentuk objek.

  • Shear X (Geseran Sejajar Sumbu X): Saat Anda memilih "Shear X" dan mengatur "Faktor Shear X (shx)", Anda akan melihat objek terdistorsi secara horizontal. Setiap titik (x, y) pada objek asli akan diubah menjadi (x + shx * y, y). Pergeseran horizontal setiap titik bergantung pada nilai y-nya dan faktor shear shx. Semakin besar nilai absolut shx, semakin besar distorsinya.
  • Shear Y (Geseran Sejajar Sumbu Y): Saat Anda memilih "Shear Y" dan mengatur "Faktor Shear Y (shy)", objek akan terdistorsi secara vertikal. Setiap titik (x, y) pada objek asli akan diubah menjadi (x, y + shy * x). Pergeseran vertikal setiap titik bergantung pada nilai x-nya dan faktor shear shy.

Perhatikan bagaimana bentuk persegi asli dapat berubah menjadi jajaran genjang akibat transformasi shear. Eksplorasi dengan nilai faktor shear yang berbeda (positif dan negatif) akan memberikan pemahaman yang lebih mendalam tentang efek geseran ini.

Contoh Perhitungan

Misalkan kita memiliki titik (3, 2) dan ingin melakukan transformasi:

Contoh 1: Refleksi terhadap sumbu X

Menggunakan matriks [ 1 0; 0 -1 ]:

x' = 1 × 3 + 0 × 2 = 3

y' = 0 × 3 + (-1) × 2 = -2

Hasil: titik (3, 2) menjadi (3, -2)

Contoh 2: Shear-Y dengan faktor shy = 0.5

Menggunakan matriks [ 1 0; 0.5 1 ]:

x' = 1 × 3 + 0 × 2 = 3

y' = 0.5 × 3 + 1 × 2 = 3.5

Hasil: titik (3, 2) menjadi (3, 3.5)
